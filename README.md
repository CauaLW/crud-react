## :clipboard: Sobre o Projeto

Um sistema de crud de usuários simples, usando JSON-server para simplificação de database e foco no frontend.

## :computer: Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- ReactJs
- React Router
- Bootstrap
- JSON-server
- Axios
- Font-awesome

## :information_source: How To Use

- Clone este repositório: `git clone https://github.com/CauaLW/crud-react`
- Vá até o repositório: `cd crud-react`
- Entre no backend: `cd backend`
- Instale as dependências para o backend: `npm i`
- Inicie o backend: `npm start`
- Saia do backend: `cd ..`
- Entre no frontend: `cd frontend`
- Instale as dependências para o frontend: `npm i`
- Inicie o frontend: `npm start`
- As portas são 3000 para o frontend, e 3001 para o backend (JSON-server)
